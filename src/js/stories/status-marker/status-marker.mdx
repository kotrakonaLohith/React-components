# Status Marker

## Pending Status Marker

Default display of the `StatusMarker` component.

<StatusMarker className="mb-32" />

```tsx
<StatusMarker />
```

## Verified Status Marker

Use the `verified` prop to display the `StatusMarker` component as "verified".

<StatusMarker className="mb-32" verified />

```tsx
<StatusMarker />
```
