## Icon Buttons

The `icon` prop takes an svg. Icon will take the button's text color.

> **note:** svg's are automatically transformed to React Components by [babel-plugin-inline-react-svg](https://github.com/airbnb/babel-plugin-inline-react-svg) so svg's **must** be passed as a component

<Button
  className="mb-32 text-bright-blue-500 hover:text-bright-blue-300"
  icon={<TrashcanIcon />}
  plain
/>

```tsx
import TrashcanIcon from 'path/to/trashcan.svg'

<Button
  className="text-bright-blue-500 hover:text-bright-blue-300"
  icon={<TrashcanIcon />}
  plain
/>
```

<Button className="mb-32" icon={<TrashcanIcon />} secondary>
  Delete
</Button>

```tsx
import TrashcanIcon from 'path/to/trashcan.svg'
<Button icon={<TrashcanIcon />} secondary>
  Delete
</Button>
```

### Adding Icon Classes

Icon color can usually be set by setting the `Button` text color. However, there are times when finer control of icon classes are needed. Use the `iconClass` prop to set `className` directly on the `icon`.

> A class to size (`rect-icon` in the example below) the `icon` is required when using the `iconClass` prop

<Button
  className="mb-32"
  iconClass="text-dark-blue-600 hover:text-dark-blue-200 rect-icon"
  icon={<TrashcanIcon />}
  plain
/>

```tsx
<Button
  iconClass="text-dark-blue-600 hover:text-dark-blue-200 rect-icon"
  icon={<TrashcanIcon />}
  plain
/>
```
