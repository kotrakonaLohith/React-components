## Overview

Pagination is used to display a set of data across multiple pages rather than in a single view. It is generally used if there are more than 10 items in a set of data but that number may vary depending on the context.

---

## Usage

The pagination component should be displayed underneath the set of data and should be centered on the page.

<div className="flex justify-center mb-40">
  <Pagination onSelect={({ index, url }) => { alert(index, url) }} currentItem={0}>
    <PaginationItem url="#1">1</PaginationItem>
    <PaginationItem url="#2">2</PaginationItem>
    <PaginationItem url="#3">3</PaginationItem>
    <PaginationItem url="#4">4</PaginationItem>
    <PaginationItem url="#5">5</PaginationItem>
  </Pagination>
</div>

When there are more than 8 pages of results, 7 should be replaced by an ellipsis and 8 should be replaced by the number of the final page of results.

<div className="flex justify-center mb-40">
  <Pagination onSelect={({ index, url }) => { alert(index, url) }} currentItem={0}>
    <PaginationItem url="#1">1</PaginationItem>
    <PaginationItem url="#2">2</PaginationItem>
    <PaginationItem url="#3">3</PaginationItem>
    <PaginationItem url="#4">4</PaginationItem>
    <PaginationItem url="#5">5</PaginationItem>
    <PaginationItem url="#6">6</PaginationItem>
    <PaginationItem ellipsis>...</PaginationItem>
    <PaginationItem url="#9">9</PaginationItem>
  </Pagination>
</div>
