## MultiSubmitInput Component

All props (`label`, `id`, `name`, `placeholder`) are required. `MultiSubmitInput` uses an `Input` component under the hood. A `MultiSubmitInput` must have one or more `MultiSubmitInputOption` components as direct children.

```tsx
<MultiSubmitInput label="Label" id="id" name="name" placeholder="placeholder">
  // MultiSubmitInputOptions here
</MultiSubmitInput>
```

## MultiSubmitInput Options

The `urlParam` prop is the only required prop. The `urlParam` is used to form a search URL that the page will redirect to. Optionally an svg can be passed to the `icon` prop to display an icon on the button. The `MultiSubmitInputOption` component uses a button element under the hood.

> **note:** svg's are automatically transformed to React Components by [babel-plugin-inline-react-svg](https://github.com/airbnb/babel-plugin-inline-react-svg) so svg's **must** be passed as a component

```tsx
<MultiSubmitInputOption urlParam="" icon={<PagesIcon />}>
  Search the site
</MultiSubmitInputOption>
```

## Keyboard Accessibility

- Submit options dropdown appears after starting to type.
- The ↑ and ↓ arrows can be used to navigate the submit options.
- `tab` and `shift + tab` function the same way as the ↑ and ↓ arrows.
- `esc` will clear the value and close the options dropdown.
